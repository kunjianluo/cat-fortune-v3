# Audio Assets

Place V3 audio assets in `assets/audio/bgm/` and `assets/audio/sfx/`.

Suggested formats:

- `.mp3` or `.ogg` for BGM and ambience loops
- `.wav`, `.mp3`, or `.ogg` for short SFX

Example filenames:

- `bgm/rain-ambience-loop.mp3`
- `sfx/door-bell.wav`
- `sfx/click.wav`
- `sfx/sacrifice-bell.wav`
- `sfx/success-chime.wav`
- `sfx/failure-slip-tear.wav`

Missing audio should be treated as optional and fall back to silence.

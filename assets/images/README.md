# Image Assets

Place V3 image assets in the category folders under `assets/images/`.

Suggested formats:

- `.webp` for backgrounds and result cards
- `.png` for transparent icons and character stills
- `.svg` only for simple UI shapes or scalable ornaments

Example filenames:

- `opening/opening-rainy-street-bg.webp`
- `cat-master/cat-master-idle.png`
- `shops/shop-bg-noodle-night.webp`
- `ingredients/ingredient-icons-placeholder.png`
- `results/result-food-card-placeholder.webp`

Missing assets should fall back to the existing CSS, emoji, or text placeholders.

# Sprite Assets

Place V3 sprite assets in `assets/sprites/cat-master/` and `assets/sprites/shopkeepers/`.

Suggested formats:

- Transparent `.png` or `.webp` for static sprite states
- Numbered `.png` sequences for lightweight frame animation
- Sprite sheets only when documented in the asset manifest notes

Example filenames:

- `cat-master/cat-master-idle.png`
- `cat-master/cat-master-judging-0001.png`
- `shopkeepers/shopkeeper-noodle-idle.png`
- `shopkeepers/shopkeeper-dessert-idle.png`
- `shopkeepers/shopkeeper-tea-idle.png`

Missing sprites should fall back to CSS, emoji, or text placeholders.
